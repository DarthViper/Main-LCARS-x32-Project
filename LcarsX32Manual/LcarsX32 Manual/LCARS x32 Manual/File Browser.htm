FILE EXPLORER

Clicking the “My Computer” button in the Start Menu will enable the LcarsX32 File Explorer. The initial view will be a list of all the drives on your computer, color-coded according to their current status. Off line drives will be shown in one color and active drives in another.

Clicking an inactive drive won’t do anything but if you click an active drive the contents will be displayed and extra action buttons will appear in the menu on the right.

PROPERTIES Button: Click and hold (0.5 seconds or longer) a folder or file to activate it. It will change color to indicate that it is in an active mode. Next, click the properties button and details about the file will appear on screen. Click the Close button to return to the drive content screen. Click any empty area in the display to de-activate the file or folder.

UP A DIRECTORY Button: Clicking this button moves the display backwards through sub folders.

OPTIONS Button: Used to access the options menu where you can set up links to other drives, or folders.

Refresh Button: Used to reload the current folder. Click this button if a file is not showing up in the current view.

GO TO Button: Clicking this button brings up a sub menu of drives and desktop links so that you can quickly move from one to another in just two clicks. The links are set up using the option menu. To set up a link click the options button then click Manage Shortcuts. Click Add then enter a name for your shortcut such as E DRIVE then click OK. Enter the path then click OK. The new shortcut will be added to the list.

You can move the shortcuts up and down the list by clicking a shortcut then clicking the Move Up or Move Down buttons.

EDIT Button: This button bring us a menu of other buttons including rename, copy, paste, cut and delete. These are all standard file explorer features and the purpose of each button is self-explanatory.

NEW FOLDER Button: Click this button to create a new folder. Enter the folder name then click OK to create it.

OPEN WITH Button: If a file has been selected, clicking this button will bring up a menu from which you can choose an application with which to open the selected file type.
